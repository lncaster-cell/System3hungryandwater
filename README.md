# System3hungryandwater — clean start

Репозиторий полностью очищен от старых данных и подготовлен для разработки новой системы механик с чистого листа.

## Что сделано
- Удалены старые служебные заглушки и артефакты прошлого состояния.
- Добавлена базовая структура для нового этапа разработки.
- Настроены правила игнорирования временных и локальных файлов.

## Стартовая структура
- `src/` — основной исходный код будущей системы.
- `docs/` — проектная документация и описание механик.
- `tests/` — тесты новой системы.
- `tools/` — вспомогательные скрипты и утилиты.
